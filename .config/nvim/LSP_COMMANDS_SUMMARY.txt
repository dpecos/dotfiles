WHAT WAS ADDED
--------------

Three new custom LSP commands to replace nvim-lspconfig functionality:

1. :LspInfo
   - Displays comprehensive LSP client information
   - Shows attached clients, capabilities, and configured servers
   - Opens in split window, press 'q' to close
   
2. :LspLog  
   - Opens LSP log file for debugging
   - Read-only mode, jumps to end of file
   - Press 'q' to close
   
3. :LspRestart
   - Restarts all LSP clients for current buffer
   - Automatically re-enables servers
   - Shows notifications during process

IMPLEMENTATION DETAILS
---------------------

- Location: lua/plugins/lsp.lua (lines 426-622)
- Dependencies: NONE (uses only native Neovim 0.11+ APIs)
- Native APIs used:
  * vim.lsp.get_clients()
  * vim.lsp.get_log_path()
  * vim.lsp.stop_client()
  * vim.lsp.enable()
  * vim.api.nvim_create_user_command()
