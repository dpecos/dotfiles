🎯 COMPLETION KEYMAPS (Insert Mode)
────────────────────────────────────────────────────────────────────
  <C-Space>    Manually trigger completion
  <Tab>        Next completion item
  <S-Tab>      Previous completion item  
  <CR>         Accept completion
  <C-e>        Close completion menu
  <C-n>        Next item (traditional)
  <C-p>        Previous item (traditional)

🔧 SNIPPET NAVIGATION (Insert/Select Mode)
────────────────────────────────────────────────────────────────────
  <C-f>        Jump to next snippet field
  <C-b>        Jump to previous snippet field

📍 LSP KEYMAPS (Normal Mode)
────────────────────────────────────────────────────────────────────
  gd           Go to definition
  gD           Go to declaration
  gdv          Go to definition (vertical split)
  grr          List references (Telescope)
  gri          List implementations (Telescope)
  gO           Document symbols (Telescope)
  grn          Rename symbol
  gra          Code actions
  K            Show documentation
  <leader>k    Show signature help
  <leader>h    Toggle inlay hints
  [e           Previous error
  ]e           Next error
  [w           Previous warning
  ]w           Next warning

💡 USAGE TIPS
────────────────────────────────────────────────────────────────────
  1. Completion triggers automatically as you type
  2. Use Tab for quick navigation
  3. All LSP features work the same as before
  4. No icons = cleaner, faster interface
  5. Snippets are fully supported natively
